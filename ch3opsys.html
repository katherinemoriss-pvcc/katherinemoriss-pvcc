Operating System Terminology
Terms Definitions
Operating System Manages hardware and software resources. Provides common services for computer programs.
Job Queue Jobs waiting in mass storage to be executed.
FIFO First in first out. The order in which a job is received and executed.
Interactive Processing A program that carries on a dialogue with the user through remote terminals.
Time Sharing Operating system that provide service to multiple users at the same time.
Multiprogramming Time devided into inervals. The execution of each job is restricted to one job at a time..
Application Software Programs for performing tasks particular to the machine.
Ex: Spreadsheets, database systems, etc.
System Software Performs tasks common to computer systems in general.
Provides infrastructure that application software requires.
Utility Software Programs that perform activities that are fundamental to computer installations.
Consists of software units that extend the capabilities of the Operating System.
Graphical User Interface (GUI) Allows commands by using one of several input devices (mouse, keyboard).
Files as programs are displayed as icons on the monitor.
Shell Older user interface. Communicated with users through textual message using a keyboard and monitor.
Kernel Internal part of Operating System that contains software components
to perform bacic functions required by computer installation.
Device Driver Component of the kernel. Software units that communicate with the controllers (peripheral devices)
to carry out operations on the device attatched to the machine.
File Manager Coordinates the use of the machines mass storage facilities. Maintains records of files stored in mass storage.
Memory Manager Coordinated the machines use of mass memory. Component of the Operating System kernel.
Virtual Memory "Fictional" memory space created by "paging"
Boot Strapping Transfers an Operating System from mass storage to memory.
ROM Read Only Memory. Contents can be read not altered. Non-volitile memory cells.
Deadlock Two or more processes are blocked from progressing.
Spooling Technique for holding data for output at a later, more convenient date.
Super User (Administrator) Establishes accounts on a computer. Can alter Operating System settings.
Multi Factor Authentication Requiring more than one piece of evidence (personal) to gain access to an account.
