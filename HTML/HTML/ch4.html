Item Definition
Network Classifications
LAN-Local Area Network A set of computers located in a single building or complex
MAN-Metropolitan Area Network Spans a community. Intermediate size.
WAN-Wide Area Network Links over great distances like neighboring cities or across the world.
Network Models
Bus Network Machines connected to a common communication line, aka a bus.
Star Network Machine serves as a focal point to which all other machines are connected.
Protocols
CSMA/CD (Ethernet) Multiple access with collision detection. Each message must be broadcast to all machines on the bus.
Not compatable with a star network.
CSMA/CA (WiFi) Multiple access with collision avoidance; avoid rather than detect collisions.
Combining Networks
Repeater Passes signals back and forth between two original buses.
Bridge Connects two busses but does not necissarily pasll all messages across the connection.
Switch A bridge with multiple connection that can connect to more than two busses.
Router Connection between networks to form Internet. Special purpose computers used for forwarding messages.
Gateway Point in which a computer is linked to the Internet. Serves as a passageway between a netork and the outside world.
Client/Server Model
P2P-Peer to Peer Involves proccesses that provide service to and recieve service form each other. Execute on temporary basis.
Distributed System Software units that process on different computers.
Cloud Computing Distributed system. Huge pool of shared computers on the network. Can be allocated for use by clients as needed.

          Internet Four Layer Software
          Application Layer Where a message originates and prepares for the destination address
          Transport Layer Chops Messages into packets
          Network Layer Assigns intermediate address to each packet
          Link Layer Transfers packet.
          Internet IS
          Pocket Switching Network Groups data into short messages in a fixed format transmitted over digital network.
          Fault Tolerant System maintains proper operations despite failures or
          faults in one or more components.
          Redundant Practice of incorporating backup components or systems to ensure a computers uninterrupted operation.
          Cyber Security
          Malware
          • Malicious Software that attacks a computer from a distance.
          • Can be transferred to and executed on the computer itself
          Virus
          • Software that infects a computer
          • Inserts itself into programs already residing within the computer.
          • As host program is started the virus becomes active.
          Worm
          • Autonomous program that transfers itself through a network.
          • Takes up residency in a computer duplicating itself.
          • Forwards copies of itself to other computers.
          Trojan Horse
          • Enters program as a game or useful utility program.
          • Can lie dormant until triggered.
          • Performs harmful effects on a computer.
          Spyware
          • "Sniffing Software".
          • Malicious software that collects information about the activities on a computer.
          • Reports information back to the investor of the attack.
          Phishing Obtains information explicitly by asking for it, hoping victim takes the "bait".
          DDOS-Distributed Denial of Service Attack Works by generating attacks over many hosts using unsuspected computers.
          Protection
          Firewall Primary technique that filters traffic passing through a point in a network.
          Spam Filter Blocks unwanted mail.
          Proxy Server
          • A software unit that acts as an intermediary between the client and server.
          • Goal: Sheild client from adverse actions of the server.
          Antivirus Software Detects and removes the presence of known viruses and other infections.
          Crytography The study of sending and recieving messages securely in the presence of adversaries.
          Public Key Encryption Technique of encryption/decryption algorithms known but encoded message remains secure.
